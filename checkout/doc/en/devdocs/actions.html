Actions

  1. An action must be defined in the Enumeration "ACTIONS"
  2. Listen:
     
    class xxx implements ActionCallBackListener {

      ...

      ActionManager.addActionCallbackListener( ACTIONS.VIEW_FULL, this );
      ...

      public boolean handleAction( ACTIONS action, ActionObject o ) {
        switch ( action ) {

          case CHANGE_LANGUAGE:
            setTexts();
            break;

          default:
          break;
        }

        return true;

      } // handleAction
    } // class xxx

    Important note:
    The return value must be true to make other classes, that listen to the same action, perform their reactions. If return value here is false, no other class will wake up on that action anymore!
     

  3. The action must be fired somewhere to make all listeners react to is. Fire the action like this:
     
    ActionManager.getAction( ACTIONS.CHANGE_LANGUAGE ).execute( null );

    Important note:
    The parameter of the execute method takes an object. You can use that object while handling the action!
