Getting started

  1. First of all, a new developer must download eclipse (of course anything else can be used, too. But we strongly recommend eclipse as it is perfectly integrated for java development). This documentation will refer to eclipse users.
    You can get Eclipse from www.eclipse.org. Make sure to get the latest version available.
  2. To access the Subversion repository on sourceforge, there is a special plugin for eclipse needed, called "subclipse". Follow the instructions on subclipse.tigris.org to install the plugin into eclipse.
  3. Open eclipse
  4. Go to Window --> Open Perspective --> Other
  5. Open SVN Repository Exploring
  6. Switch to perspective SVN Repository Exploring
  7. On the left you see the tab SVN Repository. Open the context menu there (right click)!
  8. Choose New --> Repository Location... there and enter the URL for the battleforge project:
    https://svn.sourceforge.net/svnroot/battleforge
    If everything is configured correctly, a new entry will appear for the SVN repository that can be opened now
  9. Open the trunk node now
  10. With a rightclick onto the entry de.battleforge there should be an entry called checkout. Click this entry and choose your workspace
  11. After a few minutes all the files will be downloaded and you can switch back to the java perspective where you should see all the files of the battleforge project waiting for you to work with them...
  12. You're done
