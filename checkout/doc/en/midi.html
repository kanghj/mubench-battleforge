Improving of sound qualität (Midi)

With Java there is normaly a standard soundbank for output of midi-sounds provided. There are however better soundbanks available for download from Suns website. If backgroundmusic should be played, it might be interesting to use those banks to improve sound quality. To achiev this, you have to follow the following steps:
  • Download the alternative soundbank:
    http://java.sun.com/products/java-media/sound/soundbank-deluxe.gm.zip
    (If the link is not valid, try searching the site for keyword "soundbank-deluxe")
  • Extract the contents of the archive to a local folder and copy the files to <Java-Runtime>\lib\audio\.
  • or extract the files directly to <Java-Runtime>\lib\audio\.

Attention:
The files must be copied into the runtime environment that is actually used to run BattleForge to hear the difference. If there are other runtimes installed (e.g. the JDK (Java Development Kit)), you should find out, which one is used! For the normal user this hint isn't important because normally there should be only one runtime installed (see Minimum Requirements).
