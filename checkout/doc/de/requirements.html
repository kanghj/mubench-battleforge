Anforderungen

Client

  • Java Runtime-Evironment (Version 1.5.0 oder höher) muss installiert sein
  • Java-Webstart muss verfügbar sein (ist normalerweise da nach der Installation)
  • Die Bildschirmauflösung sollte nicht kleiner als 1024x768 Bildpunkte sein

Server

  • PHP5 oder höher
  • mySQL 5.x oder höher
